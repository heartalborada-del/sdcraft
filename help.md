# 帮助

## 命令帮助

/spawn 回主城  
/home 回家  
/sethome 设置家  
/tpa {player} 请求传送到其他玩家那儿  
/tpaccept 确认传送请求  
/tpdeny 拒绝传送请求  
/tpacancel 取消你发送的传送请求  
/m {player} {msg} 或者是 /msg {player} {msg} 给别人发送悄悄话  
/afk 开启暂离模式  
/res create {res\_name} 创建领地  
/res help 获取领地创建帮助  
/jukebox 打开音乐盒
